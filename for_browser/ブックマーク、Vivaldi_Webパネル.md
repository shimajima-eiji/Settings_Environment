# ブックマーク、Vivaldi_Webパネル
Webパネルと拡張機能、検索エンジンを設定する。
<br>スタンドアロンが必要なものもあるので別途インストールすること。

# Webパネルに入れるもの
アカウントに紐付いたりインポート出来ないので、諦めて手動で追加する。  
Webパネルに入れず、ブックマークに入れるのも一手か。
